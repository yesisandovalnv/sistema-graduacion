### 🔴 PROBLEMAS CRÍTICOS (Bloqueantes)
```
1. JWT en localStorage (XSS vulnerable)      → 2h fix
2. Sin rate limiting (fuerza bruta)          → 1h fix
3. DEBUG=True por default                    → 30m fix
4. SECRET_KEY en repo                        → 1h fix
5. Sin HTTPS en producción                   → 2h fix
─────────────────────────────────────────────
Total corrección: 6.5 horas
```

### 🟠 PROBLEMAS ALTOS (Recomendados)
```
6. N+1 queries en dashboard                  → 3h fix
7. Sin validación de archivos                → 2h fix
8. Sin logging centralizado                  → 3h fix
9. Status codes inconsistentes               → 2h fix
10. Sin timeout en requests                  → 1h fix
─────────────────────────────────────────────
Total corrección: 11 horas
```
